# 프로젝트 소개
==========

### 브랜드 스토리
----------
핵심 키워드 - “가치”
부모가 아이를 키우며 하는 안타까움과 애뜻함이 담긴 말 '에쿠에쿠'가 모티브가 되어 친환경 '에코'와 결합되었으며 부모의 마음으로 펀딩 프로젝트 진행을 돕고 실패한 펀딩은 업사이클링 하여 더 좋은 기회를 만들어 줍니다.
